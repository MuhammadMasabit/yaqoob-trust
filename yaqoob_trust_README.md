## ❤️ About Yaqoob Trust

**Yaqoob Trust — Ray of Hope** is a volunteer-driven social welfare society dedicated to:

- 🩸 **Blood Donation** — Emergency blood arrangement 24/7
- 🤝 **Helping Needy Persons** — Food, clothes, and essentials
- 💝 **Social Donations** — 100% transparent welfare work
- 🌟 **Community Building** — Growing volunteer network
